# Deferred Hosted And Social Platform

Status: deferred as of 2026-06-09.

This document preserves ideas for a possible future phase:

- accounts
- encrypted multi-device sync
- public profiles
- friends
- hosted share pages
- verified achievements
- leaderboards
- anti-cheat activity sessions
- hosted source health reporting

No active extension module may depend on these features.

Before this work begins, create a separate approved plan covering:

- hosting and cost
- privacy and legal requirements
- moderation
- account security
- data deletion
- anti-cheat limitations
- operational ownership

The extension must remain useful offline if this platform is never built.
